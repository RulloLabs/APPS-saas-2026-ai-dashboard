# SaaS 2026 AI Dashboard | Production Ready

This is a state-of-the-art SaaS dashboard built with Next.js (App Router), Tailwind CSS, and AI-first engineering principles.

## Features
- **Adaptive Density**: Toggle between standard and compact views for maximum productivity.
- - **AI Insights**: Built-in AI layers for real-time business analysis and recommendations.
  - - **Command Bar**: Global shortcut-driven interface (Ctrl+K) for rapid navigation and actions.
    - - **Edge Powered**: API endpoints optimized for the edge for ultra-low latency.
      - - **Modern UI**: Dark-mode first design with Outfit/Inter font pairing and glow effects.
       
        - ## Tech Stack
        - - **Framework**: Next.js 15+ (App Router)
          - - **Styling**: Tailwind CSS / Lucide React
            - - **Infrastructure**: Vercel (Edge Functions) / GitHub Actions
              - - **Database**: Supabase Ready
                - - **AI**: OpenAI SDK Ready
                 
                  - ## Structure
                  - - `/app`: Main application routes and page layouts.
                    - - `/components`: Reusable UI components including the `CommandBar`.
                      - - `/ai`: Core AI logic for analyzing, refactoring, and generating content.
                        - - `/api`: Specialized edge-runtime API endpoints.
                         
                          - ## Getting Started
                          - 1. Clone the repository.
                            2. 2. Run `npm install`.
                               3. 3. Set up your environment variables (`.env`).
                                  4. 4. Run `npm run dev`.
